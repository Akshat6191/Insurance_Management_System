# Insurance_Management_System
A C++ based system to manage insurance policies, clients, and claims. Basic insurance management system built in C++ for tracking policies and customers. A simple console-based app to manage insurance records using C++.
Author-Anuj Knawar , Divyanshu Sharma , Akshat Singh Mankotia


### Complete Concept Map
```
AbstractPolicy.h   → Abstract Class, Pure Virtual, Virtual Destructor,
                     Virtual Functions, this Pointer, Default + Parameterized
                     + Copy Constructors, Namespace

Encapsulation.h    → Encapsulation, Private Data, Getters/Setters,
                     Inline Functions, Friend Function, Copy Constructor,
                     this Pointer, Access Specifiers, Destructor

Inheritance.h      → Single Inheritance, Multilevel Inheritance (3 levels),
                     Multiple Inheritance, Mixin Class, Constructor Chaining,
                     Default Arguments, override keyword, Destructor

Polymorphism.h     → Runtime Polymorphism, Function Overloading,
                     Operator Overloading (+, >, <, ==, <<),
                     Static Members, Base Class Pointer

Templates.h        → Template Class, Template Functions,
                     Static Data Members, Static Functions,
                     Exception Handling (try/catch/throw),
                     Dynamic Memory, References, Default Arguments

main.cpp           → Menu Interface, Dynamic Memory (new/delete),
                     Namespace usage, All concepts demonstrated live
